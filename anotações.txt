#NPM

Documentação links:
https://docs.npmjs.com/misc/
https://docs.npmjs.com/misc/config

Configurando o NPM:
- npm set init-author-name "Andrey Frazatti Alves"
- npm set init-author-email "pairsoftbusiness@gmail.com"
- npm set init-author-url "https://github.com/PairSoftGithub"
- npm set init-license "MIT"


Criando o GitIgnore
- npm install gitignore -g
- gitignore -types
- gitignore Node


Logando no NPM:
- npm adduser

Inicializar a lib:
- npm init
- npm init -y

#GIT

Comandos:
- git config --global user.name "Andrey Frazatti Alves" (nome de usuário).
- git config --global user.email "pairsoftbusiness@gmail.com"

- git status (Verificar arquivos modificados ou adicionados)
- git add <nome do arquivo> ou . para todos (para adicionar os arquivos)
- git commit -m "Mensagem" (para enviar arquivos para o repositório local)
- git push (enviar para o repositório remoto)
